Privacy

FlagLint is designed as a local static-analysis CLI.

CLI data handling

The FlagLint CLI reads source files from paths you provide and writes reports or transformed files locally. It does not send source code, flag keys, file paths, LaunchDarkly SDK keys, or migration reports to FlagLint servers.

Website data handling

The website provides static documentation and links to GitHub and npm. The homepage does not include an email signup form or collect project source data. External sites such as GitHub, npm, and Cloudflare may process requests under their own policies when you visit or interact with them.

Contact

For privacy or security questions, use GitHub Security Advisories for vulnerabilities or GitHub Issues for general project questions.
