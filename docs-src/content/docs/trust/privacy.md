FlagLint runs locally in your repository. The CLI does not upload source code, flag keys, reports, or migration results to FlagLint-owned infrastructure.

The public website is static. For the hosted privacy page, see:

- [flaglint.dev/privacy.html](https://flaglint.dev/privacy.html)

## Local Reports

Reports generated by `scan`, `migrate`, and `validate` stay wherever you write them. Treat generated reports as source-derived artifacts because they may include file paths and flag keys.
